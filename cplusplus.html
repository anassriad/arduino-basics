Introduction to C/C++

Arduino sketches are written in a simplified version of C/C++. Understanding some basic concepts of C/C++ will help you write better Arduino code.

Key Concepts

  • Variables and Data Types: Learn about int, float, char, boolean, and String.
  • Functions: Functions let you organize your code into reusable blocks. Every Arduino sketch has setup() and loop() functions.
  • Control Structures: Use if-else statements and loops (for, while) to control the flow of your program.
  • Preprocessor Directives: Learn about #define for creating constant values that can be used throughout your code.
  • Comments: Use // for single-line comments and /* ... */ for multi-line comments.

Example Code

#include <Arduino.h>

// Define constants
#define LED_PIN 13

void setup() {
  // Initialize the LED pin as output
  pinMode(LED_PIN, OUTPUT);
}

void loop() {
  // Blink the LED
  digitalWrite(LED_PIN, HIGH);
  delay(1000);
  digitalWrite(LED_PIN, LOW);
  delay(1000);
}

This simple example blinks an LED connected to pin 13. Notice how the structure of the code follows the C/C++ style with functions and preprocessor directives.
